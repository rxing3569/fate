## 學習重點

化星就像四種特殊標記，提醒我們哪些人生主題會被放大、推動、重視或反覆學習。

- 四化分別為：化祿、化權、化科、化忌。
- 它不是新的星曜，而是星曜產生的變化狀態。
- 要搭配宮位與星曜一起看，不能單獨下結論。
- 它幫助我們理解一個人在哪些地方容易得到資源、展現能力、被看見，或遇到課題。

## 四化是什麼？

四化是紫微斗數中非常重要的判斷系統，它們會讓命盤中的某些位置變得更明顯，也會讓某些人生主題更容易被感受到。

「化」有變化、轉化、引動的意思，它不是命盤中另外多出來的四顆星，而是某些星曜受到天干影響後，產生不同的變化狀態。
```
四化分別是：
- 化祿：像資源補給點
- 化權：像行動推進點
- 化科：像曝光與認可點
- 化忌：像課題提醒點
```
它們不是單純的好或壞，而是提醒你：哪些地方比較容易有資源，哪些地方需要主動負責，哪些地方容易被看見，哪些地方容易反覆學習。

## 化祿是什麼？

化祿代表資源、喜好、吸引力、享受感與比較順手的地方，它常常表示一個人容易在哪些人生主題中感到有興趣，或比較容易得到資源與機會。

化祿不是單純代表錢，有化祿不一定有錢，而是代表「讓人想靠近、想投入、容易產生好感」的力量。

- 如果化祿在財帛宮，可能和金錢資源比較有關；如果化祿在夫妻宮，可能代表感情中重視舒服、好感與互相吸引；如果化祿在福德宮，可能代表一個人重視享受、精神滿足或生活品質。

```
化祿可能代表：
    - 喜歡投入的地方
    - 容易得到資源的地方
    - 容易有好感與吸引力的地方
    - 比較願意花時間經營的地方
    - 讓人覺得有收穫、有滿足感的地方
```

## 化權是什麼？

化權代表行動力、掌控感、責任、企圖心、推動力與影響力，它常常表示一個人在哪些地方會比較想主導、想證明自己，或需要承擔責任。

化權不是單純代表權力，而是代表「把事情推動起來」的力量，它會讓某個人生主題變得更有動能。  

- 如果化權在官祿宮，可能代表事業上比較有企圖心；如果化權在夫妻宮，可能代表關係中比較容易想主導；如果化權在命宮，可能代表個人行動力強，做事不太喜歡被牽著走。

```
化權可能代表：
    - 想主導的地方
    - 有行動力的地方
    - 需要承擔責任的地方
    - 容易展現企圖心的地方
    - 會想掌控結果的地方
    - 容易和權力、責任、決策有關的地方
```

## 化科是什麼？

化科代表名聲、形象、學習、修飾、秩序、方法與被認可，它常常表示一個人在哪些地方比較容易被看見，或會希望用比較漂亮、合理、有條理的方式呈現。

化科也和學習、專業、名譽、證照、口碑有關。

- 如果化科在官祿宮，可能代表工作上重視專業形象；如果化科在父母宮，可能代表容易和師長、制度、學歷、資格有關；如果化科在夫妻宮，可能代表感情中重視體面、溝通方式與外在觀感。

```
化科可能代表：
    - 容易被看見的地方
    - 重視形象與口碑的地方
    - 想要表現得有條理的地方
    - 適合學習、整理、修飾的地方
    - 容易透過專業或名聲被認可的地方
    - 比較在意是否合乎規矩或體面
```

## 化忌是什麼？

化忌代表執著、壓力、卡點、反覆學習、放不下與需要面對的課題，它常常表示一個人在哪些地方特別在意，也容易在那個人生主題中產生壓力或糾結。

化忌不是單純的壞事，它更像是命盤中提醒你「這裡需要學習」的位置，這個地方可能比較容易卡住，需要更多練習、理解與調整。

- 如果化忌在財帛宮，可能代表對金錢很在意，容易有財務壓力感；如果化忌在夫妻宮，可能代表感情中容易執著、糾結或害怕失去；如果化忌在官祿宮，可能代表事業上容易有壓力，但也可能因此很努力。

```
化忌可能代表：
    - 特別在意的地方
    - 容易有壓力的地方
    - 容易反覆遇到類似問題的地方
    - 容易想太多或放不下的地方
    - 需要學習放鬆、調整或面對的地方
    - 人生中比較有功課感的位置
```

## 四化怎麼看？

1. 先看是哪一個四化
2. 再看落在哪個宮位
3. 再看是哪一顆星曜化

對新手來說，先不用背每顆星曜的四化細節，只要知道：四化要搭配「宮位」和「星曜」一起看，才會完整。
