## 學習重點

三方四正是紫微斗數判讀宮位時的重要框架，所謂「四正」包含本宮與對宮，「三方」則是本宮加上兩個三合宮。

實務解盤不只看單一宮位，而是把這四個宮位一起看，判斷同一組人生議題如何互相支援、牽制與放大。

## 十二宮的對宮關係
| 本宮   | 對宮   | 代表含義   |
| ------ | ------ | ---------- |
| 命宮   | 遷移宮 | 自我與外界 |
| 兄弟宮 | 交友宮 | 同輩與人脈 |
| 夫妻宮 | 官祿宮 | 感情與事業 |
| 子女宮 | 田宅宮 | 延續與家庭 |
| 財帛宮 | 福德宮 | 金錢與滿足 |
| 疾厄宮 | 父母宮 | 身心與權威 |


## 常見三方四正組合
命宮的三方四正：

| 宮位   | 代表                         |
| ------ | ---------------------------- |
| 命宮   | 自我、個性、人生主軸         |
| 財帛宮 | 金錢觀、資源、賺錢方式       |
| 官祿宮 | 事業、工作、職涯方向         |
| 遷移宮 | 外部環境、外出發展、外界舞台 |

這組可以想像成：**我是誰、我怎麼賺錢、我怎麼工作、我在外界如何發展**。

看一個人的整體人生格局時，不只看命宮，也會一起看財帛、官祿、遷移。

夫妻宮的三方四正：

| 宮位   | 代表                 |
| ------ | -------------------- |
| 夫妻宮 | 感情、伴侶、親密關係 |
| 福德宮 | 內在滿足、精神狀態   |
| 遷移宮 | 外部互動、社交舞台   |
| 官祿宮 | 事業、責任、人生方向 |


這組可以想像成：**我如何愛人、我內在是否滿足、我在外界如何互動、我的事業與責任如何影響關係。**

所以感情不能只看夫妻宮，精神狀態、外部環境與事業壓力，也會影響親密關係。

財帛宮的三方四正：

| 宮位   | 代表             |
| ------ | ---------------- |
| 財帛宮 | 金錢、收入、理財 |
| 命宮   | 個人能力與性格   |
| 官祿宮 | 工作方式與事業   |
| 福德宮 | 價值感與享受方式 |

這組可以想像成：**我如何賺錢、我是什麼樣的人、我靠什麼工作、我為什麼想要錢。**

所以看財富時，不只看金錢來源(財帛宮)，也要看能力、事業模式與內在價值感。


官祿宮的三方四正：

| 宮位   | 代表                 |
| ------ | -------------------- |
| 官祿宮 | 事業、工作、職涯     |
| 命宮   | 自我特質與能力       |
| 財帛宮 | 收入與資源           |
| 夫妻宮 | 合作、關係、責任牽動 |

這組可以想像成：**我如何工作、我本身適合什麼、工作如何帶來收入、關係與責任如何影響事業。**

所以看事業時，不能只看工作本身(官祿宮)，個人能力、財務模式與關係狀態，也會牽動職涯發展。
