## 學習重點

天干地支可以理解成古代用來記錄時間與能量變化的「時間編碼系統」。
它由兩個部分組成：
- 天干：共 10個；屬於外在能量、表現出來的狀態
- 地支：共 12個；屬於內在環境、季節與時間背景
兩者依照固定順序搭配，形成 60 組不同組合，稱為「六十甲子」。

## 天干的基本概念

- 天干共有十個，所以也稱為「十天干」。
- 十天干分別是：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
- 對應五行：木、火、土、金、水；每種五行又分成陰陽兩種狀態，剛好對應十天干。

| 天干 | 陰陽五行 | 形象比喻           |
| --- | ------- | ----------------- |
| 甲  | 陽木     | 大樹、樹幹、向上生長 |
| 乙  | 陰木     | 花草、藤蔓、柔軟延展 |
| 丙  | 陽火     | 太陽、明亮、外放炙熱 |
| 丁  | 陰火     | 燈火、燭光、細緻溫暖 |
| 戊  | 陽土     | 高山、大地、厚重穩定 |
| 己  | 陰土     | 田園、泥土、孕育包容 |
| 庚  | 陽金     | 鋼鐵、刀劍、堅硬果斷 |
| 辛  | 陰金     | 珠玉、首飾、精緻修飾 |
| 壬  | 陽水     | 大海、江河、流動奔放 |
| 癸  | 陰水     | 雨露、霧氣、細膩滋養 |

天干比較像「外在能量」，偏向表現出來、容易被看見的狀態，它像是天氣，例如：太陽很強、風很大、雨水很多、空氣乾燥。

## 地支的基本概念

- 地支共有十二個，所以也稱為「十二地支」。
- 十二地支分別是：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
- 地支也會對應十二生肖與十二時辰。

| 寅  | 虎  | 03:00 - 05:00 | 陽木   | 初春破土而出的樹木。      |
| 卯  | 兔  | 05:00 - 07:00 | 陰木   | 春天柔軟旺盛的花草。      |
| 辰  | 龍  | 07:00 - 09:00 | 陽土   | 春夏交界的濕土，能承接變化。  |
| 巳  | 蛇  | 09:00 - 11:00 | 陰火   | 逐漸升起的熱氣，火勢藏而不露。 |
| 午  | 馬  | 11:00 - 13:00 | 陽火   | 正午最明亮的太陽。       |

地支比較像「內在環境」，偏向背景、基礎、季節與累積狀態，它像是土地，例如：土地肥沃、地底潮濕、山勢厚重、水氣藏在地下。

## 天干地支如何組合？
天干依序循環：甲、乙、丙、丁、戊、己、庚、辛、壬、癸。
地支依序循環：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。
兩者會一起往前排列，第一組是：甲子。

接著依序排列(列舉前十二個)：
| 順序 | 干支 | 
| -- | -- | 
| 1  | 甲子 |
| 2  | 乙丑 | 
| 3  | 丙寅 | 
| 4  | 丁卯 | 
| 5  | 戊辰 | 
| 6  | 己巳 |
| 7  | 庚午 | 
| 8  | 辛未 | 
| 9  | 壬申 | 
| 10 | 癸酉 | 
| 11 | 甲戌 | 
| 12 | 乙亥 | 

地支走完十二個後，天干會繼續循環。
兩者不斷往前搭配，直到第 60 組才會重新回到甲子。

**為什麼是 60 組，不是 120 組？**
天干有 10 個，地支有 12 個，如果任意配對，理論上會有 120 組，但天干地支不是任意配對，而是依照**固定順序**同步前進，因此最後形成 60 組，而不是 120 組。
