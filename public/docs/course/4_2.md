## 學習重點
- 時運(流運)通常指：大限、小限、流年。
- 大限：用來觀察「十年階段」的人生主題，看的是中長期人生方向。
- 小限：用來觀察「一年之內」較容易被啟動的重點，看的是短期年度變化與個人主觀感受。
- 流年：用來觀察「西元單一年度」的外在環境與客觀事件焦點，看的是當年的客觀變化與事件觸發。
- 大限、小限與流年都不能單獨判斷吉凶，需要搭配本命盤、宮位、星曜、三方四正與四化一起理解。

## 快速認識

從本命、大限、流年、小限四個層次來解釋：

| 類型   | 觀察範圍       | 比喻     | 重點                       |
| ------ | -------------- | -------- | -------------------------- |
| 本命   | 一生的基本結構 | 人生地圖 | 我原本是什麼樣的人         |
| 大限   | 十年階段       | 十年章節 | 這十年我在經歷什麼主題     |
| 小限   | 一年個人狀態   | 年度任務 | 今年哪些事情主觀上被引動與在乎 |
| 流年   | 當年外在環境   | 年度天氣 | 今年外在環境對我有何機會或考驗 |

如果本命盤像一本書，大限像其中一個篇章，小限就像這個篇章裡當年被翻到的頁面，而流年則是當年的天氣環境，影響著我們在閱讀這一頁時的客觀遭遇。


## 什麼是大限？如何看大限？

在紫微斗數中，「大限」可以理解成一個人生命中的 **十年階段**。

如果本命盤是一張完整的人生地圖，那麼大限就像是：人生每十年打開的一個章節。

```
不同的大限可能會代表：
    - 某十年比較重視事業發展。
    - 某十年比較重視感情關係。
    - 某十年比較容易思考人生方向與自我定位。
```

大限不是在說「這十年一定會發生什麼事」，而是在提醒：**這十年有哪些人生主題比較容易被啟動，哪些面向值得多觀察。**

看大限時，不要只看「目前大限落在哪一宮」就直接下結論，而是：

```
1. 找到目前所在的大限
2. 看大限命宮落在哪一宮
3. 看大限三方四正
4. 看大限中的星曜組合
5. 看大限四化
6. 疊回本命盤一起理解
```

```
大限探索問題
    - 這十年我正在學習什麼？
    - 這十年我最在意的人生主題是什麼？
    - 這十年我在哪些地方需要建立能力？
    - 這十年我是否正在改變人生方向？
```


## 什麼是小限？如何看小限？

在紫微斗數中，「小限」可以理解成**某一年**對應到命盤中的年度觀察位置。

如果大限是十年章節，那小限就像是：這一年被翻開的一頁。


```
小限通常用來輔助觀察某一年的狀態、重點與變化：
    - 今年比較容易關注哪個人生面向？
    - 今年在哪些關係或事件中容易有感受？
    - 今年適合觀察哪些調整方向？
    - 今年有哪些主題會被放大？
```

小限不是用來單獨判斷一整年的好壞，而是幫助我們看：**今年的注意力可能會落在哪裡。**

小限可以作為年度觀察的輔助工具，可以依照以下順序：

```
1. 找到今年的小限位置
2. 看小限落在哪一宮
3. 理解該宮代表的人生主題
4. 觀察宮內星曜與相關結構
5. 搭配大限與本命盤一起理解
```

```
小限探索問題：
    - 今年我最需要關注什麼？
    - 今年哪個人生面向最容易被觸動？
    - 今年我可以調整什麼習慣？
    - 今年我可以在哪些地方更有意識？
```



## 什麼是流年？如何看流年？

在紫微斗數中，「流年」代表的是**特定年度**（例如 2026 丙午年）外在環境對我們的客觀影響與事件焦點。

如果本命盤是人生地圖，大限是十年主題，流年就像是：當年天空所下的雨或刮的風（外在氣候）。

```
流年通常用來觀察當年的客觀環境變化與事件觸發：
    - 今年容易遇到什麼樣的外在機會或環境挑戰？
    - 今年外在環境的焦點在哪個生活領域（如官祿、財帛等）？
    - 今年容易觸發哪些具體外在事件（如換工作、出國、買房等）？
    - 今年需要注意哪些客觀的人際或外部環境變化？
```

流年代表的是當年的外在客觀環境，判讀流年時，可以依照以下順序：

```
1. 找到今年的流年命宮位置（流年命宮固定落在當年的地支位置，例如 2026 丙午年，流年命宮就在「午」宮）
2. 看流年命宮落在哪個本命宮位與大限宮位（進行三層疊宮判讀）
3. 觀察流年命宮內所落的星曜組合與流年四化
4. 判讀流年命宮的三方四正結構
5. 結合大限的十年背景與本命盤特質，綜合分析當年的客觀運勢
```

```
流年探索問題：
    - 今年外在環境帶給我什麼樣的機會與考驗？
    - 我該如何在今年的外在趨勢中順勢而為？
    - 今年在哪個宮位代表的領域最容易有實質事件觸發？
    - 我該如何調適今年外在氣候環境對我造成的影響？
```



## 常見誤區

- 誤區一：大限就是判斷十年好壞
- 誤區二：小限就是判斷今年一定會發生什麼
- 誤區三：只看小限就能判斷整年運勢
- 誤區四：大限不好就代表十年都不好
- 誤區五：大小限可以取代本命盤
- 誤區六：流年等同於小限（流年代表客觀環境事件，小限代表主觀內在感受，兩者有別）
- 誤區七：流年不好就代表整年都會發生災難
- 誤區八：忽略三層疊宮，只看流年宮位就急著下結論
